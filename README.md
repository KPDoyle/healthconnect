# HealthConnect

A privacy-first personal health data hub prototype.

## Features

- Health overview with wearable and clinical metrics
- Personal health timeline
- Clinical records and document imports
- Medication list
- Data-source connection centre
- Privacy, sharing and local data export controls
- Clear separation between source data and AI-generated observations

## Run locally

Open `index.html` in a modern browser or serve the repository with any static web server.

## Deployment

GitHub Pages deploys automatically from the `main` branch through `.github/workflows/pages.yml`.

> Prototype only. It does not provide medical advice, diagnose conditions, or connect to live NHS, Apple Health or wearable APIs.
